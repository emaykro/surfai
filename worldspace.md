# Project: Vibe-Ninja (Open Source Predictive Analytics)
## Vision
Autonomous performance marketing tool for real-time conversion prediction.

## Tech Stack
- Frontend SDK: Vanilla TS (Zero dependencies)
- Backend: Fastify (Node.js) on "Kukuruza" server
- ML Engine: LightGBM (Python) - Local Inference
- Database: PostgreSQL

## Agent Hierarchy
1. **Architect (Claude Code)**: High-level planning & logic.
2. **Security Sub-Agent**: Key leak prevention & SQLi checks.
3. **ML Optimizer**: Data processing & model training.
